# Super Auto Pokémon (Working Title)

A work-in-progress auto-battler inspired by *Super Auto Pets* and *Pokémon*.  
Built in Python using Pygame.

## Status
Early development. Expect bugs, placeholder assets, and frequent changes.

## About
This is a fan-made, non-commercial project created for learning and experimentation.  
It mixes the team-building/autobattler mechanics of Super Auto Pets with creatures, abilities, and progression inspired by Pokémon.
